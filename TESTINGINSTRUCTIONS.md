Steps in testing
1.  Upload a file on IPFS Desktop
2.  Grab the CID of the uploaded file
3.  Deploy the contract
4.  Fill up the file information (IPFS, file name, and the subject/course) on the UploadFile function
5.  Complete transaction
6.  Get the fileID
7.  Input the file ID in the getFile function
8.  Copy the URL after completing the transaction
9.  Paste the URL in any browser
    NOTE: If it says 504 Gateway Timeout, just refresh the page
10. File displayed in browser
