Steps in testing

NOTE: Download IPFS Desktop (It's Free) (This is for the Uploader)
1.  Open IPFS Desktop
2.  Click the "Files" on the left side bar
3.  Upload a file on IPFS Desktop using the Import Button or drag the File to the Application
4.  Click on the Three Dots on the right side of the File
5.  Copy the CID of the uploaded file
6.  Compile the Solidity Code
7.  Deploy the contract
8.  Fill up the file information (Paste the copied IPFS, enter a file name, and enter the specific subject/course) on the UploadFile function
9.  Click Transact
10.  Complete transaction
11.  Click the getCourses function and input the subject name
12.  Get the fileID by finding the File number
13.  Input the file ID in the getFile function
14.  Copy the URL after completing the transaction
15.  Paste the URL in any browser
    NOTE: If it says 504 Gateway Timeout, just refresh the page
    NOTE: You can copy the CID itself from the getFile and paste it on the IPFS Desktop to get the File (If the receiver has IPFS Desktop Installed)
16. File is displayed in the browser
